POS Dashboard

🔖 Odoo 18.0 📅 May 2026 Dashboard KPI Analytics

Advanced analytics dashboard for Point of Sale — real-time KPIs, top products, session summaries and multi-currency support.

📌 About This Module

The POS Dashboard module extends the Odoo Point of Sale with a rich analytics interface. It aggregates session data and presents actionable KPIs directly inside the POS screen, helping managers make faster decisions.

This module is compatible with Odoo 18.0 and requires no external dependencies.

ℹ️ This module does not modify any core Odoo models. It uses computed fields and SQL queries for performance.
✨ Key Features
📈
Real-time KPIs
Total sales, unique products, average order value — updated live during POS sessions.
🏆
Top Products
Ranks best-selling items by quantity and revenue with filtering by date range.
💱
Multi-Currency
Displays sales broken down by currency with live exchange rate support.
🗂️
Session Summary
Per-session breakdown of payments, discounts and cashier performance.

📦 Installation
⚠️ Make sure you are running Odoo 18.0 before installing this module.
1
Copy the module to your addons path
Place the pos_dashboard folder inside your custom addons directory.
cp -r pos_dashboard /path/to/odoo/custom-addons/
2
Update the addons list
Go to Settings → Activate Developer Mode, then Apps → Update Apps List.
3
Install the module
Search for "POS Dashboard" in the Apps menu and click Install.
🖼️ Replace with: <img src="screenshots/install-step.png">
📸 Module install screen
4
Configure POS Settings
Navigate to Point of Sale → Configuration → Settings and enable the Dashboard option under the Analytics section.
⚙️ Configuration Options
  • Date Range Filter — Set default date range shown on dashboard (Today / Week / Month).
  • Top N Products — Choose how many top products to display (default: 10).
  • Exclude Discounts — Toggle whether discount products appear in product counts.
  • Currency Display — Show amounts in company currency or original payment currency.
